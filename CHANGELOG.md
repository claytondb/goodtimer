# Changelog

All notable changes to GoodTimer.

## [1.1.0] - 2026-02-21

### Added
- **PWA Support** - Install as app on mobile devices
- Marketing improvements

### Fixed
- PWA manifest to use SVG icon
- Removed missing PNG icon references

---

## [1.0.0] - 2026-02-20

### Added
- **Initial Release**
- Timer system (5, 10, 15, 20, 30 minute options)
- Star points reward system
- Daily streak tracking
- Activity categories (Reading, Exercise, Chores, Practice, Homework)
- Rewards shop with redeemable items
- Activity history log

### Rewards
- TV Time (30 min) - 10 stars
- Video Games (30 min) - 15 stars
- Ice Cream Treat - 20 stars
- Movie Night - 30 stars
- Fun Outing - 50 stars
- Special Surprise - 100 stars

---

*A reward timer to help kids earn screen time and treats through activities.*
