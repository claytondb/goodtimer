# ? GoodTimer

> Reward timer for kids - Earn time, track rewards, be awesome!

## ? Features

- **Timer System** - Set 5, 10, 15, 20, or 30 minute timers
- **Star Points** - Earn stars for completing activities
- **Streak Tracking** - Keep your daily streak going
- **Activity Categories** - Reading, Exercise, Chores, Practice, Homework
- **Rewards Shop** - Redeem stars for treats and privileges
- **History Log** - Track past activities and rewards
